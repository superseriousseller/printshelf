How it works

  1. 1

    Log every print

    Title, photo, filament, printer, rating. Notes if you want them. Failed prints stay on the wall because pretending they didn't happen feels wrong.

  2. 2

    Track your library

    Catalog your filament spools by brand, material, color, and status. Queue the next model you want to print so you don't lose the link.

  3. 3

    Share your shelf

    Every account gets a public URL — printshelf.app/@your-handle. Show off the prints you're proud of, link it on Reddit, send it to your friends.

What's in the box

  • Print log + queue

    Two states: queued and printed. Move them between with one click. Filter the wall by material, status, or rating.

  • Filament library

    Brand, material, color, diameter, status. Tag prints with the spools you used — even multi-material prints with multiple filaments.

  • Public profile

    SEO-indexable, mobile-friendly. Visitors don't need an account to see your shelf.

  • Chrome extension

    One-click import from Makerworld, Printables, Cults3D, Thingiverse — and one-click filament save from Bambu, Polymaker, Amazon, and more. Install free →

  • Pro tier

    Unlimited prints and filaments. $4.99/mo or $39/yr. Upgrade →

  • API access

    Every account gets an API key. Build your own integrations against PrintShelf if you want.
